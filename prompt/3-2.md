
AlphaFlow US에 LSTM 기반 5일 후 방향 예측 모델을 추가해라.

■ 배경
현재 numeric_score는 RSI/MACD/SMA의 "현재 스냅샷"이다. 미래 방향을 예측하지 않는다.
Backtest 2의 win_rate ≈ 53%는 동전 던지기 수준이다.
LSTM으로 5일 후 상승/하락을 예측하여 진입 시그널 품질을 높인다.

■ 할 일

1) requirements.txt에 torch가 없으면 추가 (3-1에서 이미 추가했을 수 있음)
2) api/services/lstm_predictor.py 신규 생성

   - classStockLSTM(nn.Module)
     · Input: (batch, 60, 10) — 60일 윈도우, 피처 10개
     · LSTM: hidden=64, layers=2, dropout=0.3
     · FC: 64 → 32 → ReLU → Dropout(0.2) → 1 → Sigmoid
     · 출력: P(5일 후 상승)
   - classLSTMPredictor
     · async def prepare_data(end_date: str) -> tuple[Dataset, Dataset]
     stock_daily에서 데이터 로드.
     피처: close_pct_change, volume_ratio, rsi_14, macd, macd_signal,
     bb_percent_b, sma20/close 비율, sma60/close 비율, atr/close 비율, vix_close.
     vix_close가 stock_daily에 없으면 별도로 yfinance에서 ^VIX 로드해서 날짜 매칭.
     피처가 없는 종목/날짜는 건너뛰기.
     레이블: 5일 후 close > 오늘 close → 1, 아니면 0.
     60일 윈도우로 잘라서 Dataset 구성.
     end_date 기준 이전 = train, 이후 = val.

     · async def train(train_end_date="2024-06-30", epochs=50, batch_size=256, lr=0.001) -> dict
     학습 실행. early stopping (val loss 5 에폭 개선 없으면 중단).
     모델 저장: api/services/models/lstm_latest.pt
     반환: {train_acc, train_auc, val_acc, val_auc, overfit_gap, epochs_trained}

     · async def predict(symbol: str) -> dict
     최근 60일 데이터로 추론.
     반환: {symbol, lstm_score (prob×100, 0-100), direction ("UP"/"DOWN"),
     confidence (|prob-0.5|×2)}

     · async def predict_batch(symbols: list[str]) ->list[dict]
3) api/routers/에 LSTM 엔드포인트 추가

   - POST /api/lstm/train (body: {train_end_date, epochs}) → train 호출
   - POST /api/lstm/predict (body: {symbol}) → predict 호출
   - POST /api/lstm/predict-batch (body: {symbols}) → predict_batch 호출

■ 주의

- 백테스트 통합은 이 프롬프트에서 하지 마라. 먼저 val AUC > 0.55 확인 후 별도 통합.
- torch 추론은 동기이므로 asyncio.to_thread 사용.
- 모델 파일 경로: api/services/models/ 디렉토리 생성 필요.
