
AlphaFlow US에 FinBERT 기반 감성 분석을 추가해라.

■ 배경
Backtest 1에서 text 가중치(0.35) 적용 시 수익률 +1.85%,
Backtest 2에서 text 가중치 0 적용 시 +11.27%.
현재 키워드 기반 감성 분석이 오히려 성과를 악화시킨다.
FinBERT로 교체한 뒤 실제로 도움이 되는지 검증해야 한다.

■ 할 일

1) requirements.txt에 torch, transformers 추가
2) api/services/finbert_sentiment.py 신규 생성

   - 클래스 FinBERTAnalyzer (싱글턴 패턴, 첫 호출 시 모델 로드)
   - 모델: ProsusAI/finbert
   - asyncdefanalyze(text: str) -> dict
     반환: {sentiment, confidence, scores: {positive, negative, neutral},
     sentiment_score: positive - negative (-1~+1)}
   - asyncdefanalyze_batch(texts: list[str], batch_size=16) -> list[dict]
   - 내부에서 torch 추론은 동기이므로 asyncio.to_thread로 감싸기
3) api/services/sentiment.py 수정

   - 기존 analyze_sentiment_keywords 유지 (폴백)
   - 새 함수 asyncdefanalyze_sentiment(text, method="auto") -> dict 추가
     · method="keyword": 기존 함수 호출
     · method="finbert": FinBERTAnalyzer 호출
     · method="auto": settings 테이블의 sentiment_method 값 읽기 (기본 "keyword")
   - 기존 코드에서 analyze_sentiment_keywords를 직접 호출하는 곳은 건드리지 마라
     (호환성 유지, 나중에 전환)
4) api/services/finbert_validator.py 신규 생성

   - asyncdefvalidate_finbert_vs_keyword(days=90) -> dict
   - news_articles 테이블에서 최근 N일 기사 로드 (최대 2000건)
   - 각 기사에 대해:
     · 기존 keyword sentiment_score (이미 DB에 있음)
     · FinBERT sentiment_score (새로 계산)
     · 해당 종목의 5일 후 수익률 (stock_daily에서 조회)
   - keyword_score ↔ 5d_return 상관계수, finbert_score ↔ 5d_return 상관계수 비교
   - 반환: {keyword_corr_5d, finbert_corr_5d, improvement, sample_count,
     recommendation: "adopt_finbert" | "keep_text_zero"}
   - 기준: finbert_corr > keyword_corr + 0.03이면 adopt
5) api/routers/ 에 finbert 관련 엔드포인트 추가 (기존 라우터에 추가하거나 새 라우터)

   - POST /api/finbert/validate → validate_finbert_vs_keyword 호출
   - POST /api/finbert/analyze (body: {text}) → 단건 분석

■ 주의

- ProsusAI/finbert 모델은 ~400MB. 메모리 부족 시 Ollama 모델과 동시 로드 불가.
  FinBERT 사용 시 Ollama embed 등과 순차 실행 필요.
- 백테스트에는 아직 통합하지 마라. validate로 가치를 먼저 검증.

■ 완료 기준

- /api/finbert/analyze에 텍스트 보내면 sentiment, confidence, sentiment_score 반환
- /api/finbert/validate 호출 시 keyword/finbert 상관계수 비교 결과 반환
- recommendation 값이 정상 출력
